**Is there morphological superlative marking on the property word?**

This question asks whether there is a bound marker of superlative degree on the property word, as in Finnish example (1). Property words roughly correspond to adjectives.

(1) Finnish<br/> 
>*kaunis : kauni-impi : kaune-**in***<br/> 
>beautiful : beautiful-COMP : beautiful-SUPL<br/> 
>‘beautiful' : 'more beautiful' : 'the most beautiful’

To compare, in Erzya, respective constructions are formed by using *veśe* 'all' in the ablative case followed by an adjective in the nominatiive case (Aasmäe 2012). Thus, Erzya is coded as '0'.

**Coding.** The answer is '1' if a language has a morphological marking available to express the highest quality/degree (as in Finnish). The answer is '0' if there is no bound marker available (as in Erzya).
