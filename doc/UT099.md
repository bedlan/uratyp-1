**Is there (in addition to the basic declension) a declension different from a possessive declension?**

Nominal inflection concerns case marking. For instance, the Finnish noun  'friend' inflects as follows: *ystävä* (Nom) : *ystävän* (Gen) : *ystävää* (Prt), etc. In addition to case inflection, there may also be possessive inflection, e.g., Finnish *ystäväni* means 'my friend', *ystäväsi* 'your friend', etc. This question examines whether nouns reveal any other types of inflection. For example, in Erzya, there is a distinction between definite and indefinite inflection of nouns (1a-b).

(1) Erzya<br/>
>a. *veĺe : veĺe-ń* (indefinite inflection)<br/>
>village.NOM.SG : home-GEN.SG<br/>

>b. *veĺe-ś : veĺe-ńt´* (definite inflection)<br/>
>home-NOM.DEF.SG : home-NOM.DEF.GEN<br/>

(Aasmäe 2012)

**Coding.** The value is '1', if there is a definite inflection (as in Erzya) or any other type of nominal inflection distinct from the basic and possessive inflection. Otherwise the value is '0'.
