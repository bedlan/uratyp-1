**Can the modifier of action nominalizations in the patient role be encoded like a direct object?**

The aim of the question is to determine whether the same encoding can be used for the modifier of an action nominalization in the patient role and a direct object. In Estonian, genitive encoding is possible in both situations (see 1a and 1b). 

>(1) Estonian<br/> 
>a. ***luuletuse** kirjutamine*<br/> 
>luuletuse kirjuta-mine<br/>
>poem.GEN write-NMZ<br/>
>'a poem writing'

>b. *kirjutan **luuletuse***<br/> 
>kirjuta-n luuletuse<br/> 
>write-1SG poem.GEN<br/>
>'I'll write a poem'

**Coding.** The value is '1' if the same case marking can be used in both situations (as in the Estonian examples).
