**Does the numeral modification of a noun have an effect on its case selection?** 

This question asks whether there are morphosyntactic differences between phrases containing cardinal numerals. For example, in Finnish, the cardinal numeral 'one' requires the nominative case, whereas cardinal numerals bigger than 'one' take the partitive case (1).

(1) Finnish<br/>
>*yksi ystävä :  viisi ystävää*<br/> 
>one.NOM.SG friend.NOM.SG :  five.NOM.SG friend.PRT.SG<br/>
>‘one friend' : 'five friends’

By comparison, in Hungarian, the nominative case is used irrespective of the cardinal numeral. Thus, Hungarian is coded as '0'.

(2) Hungarian<br/>
>*egy ház : száz ház*<br/>
>one.NOM.SG house.NOM.SG : hundred.NOM.SG house.NOM.SG<br/>
>'one house' : 'hundred houses'

**Coding.** The value is '1' if a cardinal numeral can influence the case selection of its sister noun (as in the Finnish example). The value is '0' if there are no changes in the case marking irrespective of the cardinal numeral used (see the Hungarian example). The numeral 'two' was not considered if a language contained a dual number.
