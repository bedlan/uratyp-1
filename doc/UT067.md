**Is it pragmatically neutral to place an object between the negative marker and the main verb in imperative constructions?** 

The aim of this question is to determine whether an object can appear between the negative marker and the main verb in negative imperatives, as in example (1).

(1) Estonian<br/>
>*Ära **sõpru** unusta!*<br/>
>NEG.IMP friend.PL.PRT forget.CNG<br/> 
>‘Don’t forget friends’

By comparison, in Udmurt, nothing can intervene between the negative auxiliary and the main verb (example 2). Thus, Udmurt is coded as ‘0’.

(2) Udmurt<br/>
>*En das’a **s’ion**!*<br/>
>NEG  prepare.IMP:2SG  meal.NOM<br/>
>‘Don’t prepare a meal!’<br/>
>(Edygarova et al., manuscript)<br/>

**Coding.** The value is '1', if an object can appear between the negative marker and the main verb, even if this makes the sentence pragmatically unneutral. The value is ‘0’, if an object cannot intervene or if it is not pragmatically neutral anymore.
